# 📒 Introduction

**⚡ Bingstream Auto-Playlist** is a lightweight and smart script built with Node.js and Puppeteer that extracts live sports stream links directly from Bingstream.info and converts them into an auto-updating **M3U playlist**.

⏱️ The script runs automatically via **GitHub Actions**, dynamically bypassing token locks and frame protections to fetch fresh, working VIP `.m3u8` streaming links — ready to play instantly!

📡 Catch all the major live events, including **Football, Cricket, Basketball, Tennis**, and more — directly through any M3U-supported player [ NS Player, Televizo ]

> ⚠️ For educational and personal use only. Please follow the laws and regulations of your country.

# 💥 Key Features

* 🔄 Completely Automated M3U playlist generation.
* ⚡ Smart extraction that bypasses dynamic token protections.
* 🛡️ Obfuscated core script to prevent abuse and ensure long-term stability.
* 📺 Premium VIP links are automatically fetched and formatted.
* 🚀 Plug & Play ready for any standard IPTV player.
